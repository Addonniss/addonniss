v2.1.0

- Added Global Translation Lock
  * Prevents overlapping translations 
  * Ensures only one translation runs at a time
  * Eliminates duplicate API calls on rapid subtitle changes

- Refined Source Change Detection
  * Source overwrite detection now based on file size only
  * Prevents unnecessary retranslation when identical subtitles are re-downloaded

- Session-Based Translation Protection
  * Existing translated subtitles are never auto-overwritten across sessions
  * Protects user-approved translations

- Smart Mid-Session Overwrite Logic
  * Retranslation allowed only within the same playback session
  * Subtitle switch/download during playback triggers safe retranslation

- Improved Cross-Session Behavior
  * Playback restart no longer triggers unwanted retranslation
  * Prevents API cost waste
  
- Temporary translation files
   * Translations are first saved to temporary files
   * Only completed translations overwrite the final subtitles, keeping your folder clean

- Improved Debug Logging
  * Clear logging for overwrite decisions
  * Logs when translation is skipped due to active lock
