1. Download the Repository: Click here to download the Addonniss Repo
2. Open Kodi: Go to Add-ons.
3. Install from Zip: Click the box icon (Add-on Browser) and select Install from zip file. (You may need to enable "Unknown Sources" in Kodi settings).
4. Install the Addon: Once the repo is installed, go to Install from repository > Addonniss Repository > Program add-ons (or Service) > Translatarr > Install.
